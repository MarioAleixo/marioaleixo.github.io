Finally, hello world. (Days 1-4 of #100DaysOfCode)

Posted on 2017-07-06

Why ‘finally’? Because I intended to start my #100DaysOfCode on Monday, and that I did, but it took me the first four days of coding for about two hours to figure out how to set up this blog on my github.io repository.

While the process was at times frustrating, I did learn a great deal more about GitHub than I had learnt by simply reading books and guides without actually putting what I was learning in practice.

I have been coding on and off for a quite a while, but I decided to commit to completing the #100DaysOfCode challenge to take my commitment to coding to another level and see how much progress I could actually make if I put in more effort.

I can definitely say that these first four days have already taught me a good deal about Git, Github and perseverance.

To start of my 100DaysOfCode I will be working towards attaining my Front End Development certificate from freeCodeCamp. I was already halfway through the Intermediate Front End Development Projects before I stopped coding with frequency, but I’ve decided to go back and improve the all the Front End Development Projects, as well as refine my JavaScript skills, before progressing any further.

I have already redone my Perfonal Portfolio Webpage as per the fCC specs and it is available here.

The story How I got a React developer job during my 100DaysOfCode Challenge by Danny Huang featured on fCC’s Medium account is what inspired me to start coding again as seeing how much another person with not a lot of coding experience was able to achieve in 100 Days Of Code was highly motivational.

After looking into exactly what he had done to achieve as much as he did, I decided to start by following in his footsteps and blogging about my 100 Days Of Code. I found out about Hexo and how it could be used to host a blog on a Github repo, and here we are…
